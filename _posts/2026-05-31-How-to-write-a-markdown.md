## 什麼是Markdown
Markdown 是一種輕量化的標記式語言，他的副檔名是 `.md`，其用途為讓使用者專注於內容而非排版。

## 用法
- ### 標題
```md
# 一級標題
## 二級標題
### 三級標題
#### 四級標題
```
- ### 文字樣式
```md
**粗體**
*斜體*
~~刪除線~~
`行內程式碼`
```

- ### 清單
```md
  - 無序清單項目一
  - 無序清單項目二
    - 縮排子項目

  1. 有序清單一
  2. 有序清單二
  
```

- ### 連結與圖片
```md
[連結文字](https://網址)
![圖片描述](/assets/img/posts/圖片.jpg)
```

- ### 引用
```md
> 這是引用文字
> 可以多行
```

- ### 程式碼區塊
```md
# ```python
# 指定語言會有語法高亮
# def hello():
#    print("Hello World")
# ```
```
- 應該有的顯示為 （把#去掉）
```python
# 指定語言會有語法高亮
def hello():
    print("Hello World")
```

- ### 表格
```md
| 欄位一 | 欄位二 | 欄位三 |
|--------|--------|--------|
| 內容   | 內容   | 內容   |
| 內容   | 內容   | 內容   |
```
- 應該有的顯示為

| 欄位一 | 欄位二 | 欄位三 |
|--------|--------|--------|
| 內容   | 內容   | 內容   |
| 內容   | 內容   | 內容   |

- ### 分隔線
```md
---
```

## Chirpy專屬語法

- ### 圖片對齊與大小
```md
![描述](/path/to/img.jpg){: width="700" height="400" }
![描述](/path/to/img.jpg){: .left }   # 靠左
![描述](/path/to/img.jpg){: .right }  # 靠右
```

## 結語
Markdown 是一個在現在開源的程式碼時代很重要的語言，它可以讓人們簡單明瞭地知道他們應該要做什麼操作，同時提供編寫者不須煩惱怎麼排版而只須思考他們的內容怎麼呈現，可以在這篇文學到Markdown的基本文法。
